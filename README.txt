OSRS Companion v5

Open index.html in your browser.

New in v5:
- Data Pack page for one-click wiki-derived quest info import.
- Imported quest details are cached locally in your browser.
- Dashboard has progress cards, route preview, blocker radar, skill blocker graph-style bars, and data coverage.
- Quest prereqs are clickable and open inside the tracker.
- Wiki links remain visible on every quest.
- Item checklist is route-first instead of random/alphabetical.
- Tools page uses clickable tool cards.

If the Data Pack importer fails from a local file in Safari or a strict browser:
1. Open Terminal in this folder.
2. Run: python3 -m http.server 8000
3. Open: http://localhost:8000

Progress and imported data are stored in browser localStorage. Use Save / Import to back them up.
